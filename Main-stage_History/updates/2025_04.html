接待中心頁面 - 功能全面升級

計時器功能

新增完整的計時器功能，允許用戶啟動、暫停和重置計時器。

彈窗登入界面

將登入按鈕改為彈窗登入界面，直接提交登入請求，提升用戶體驗。

改進登出功能

通過 AJAX 發送登出請求，成功後自動跳轉到 usercenter.php。

修復事件綁定錯誤

修復登入輸入框事件綁定錯誤，確保元素存在時才綁定。

待辦事項清單

新增待辦事項清單功能，使用 localStorage 儲存任務。

天氣查詢功能

新增天氣查詢功能，使用 OpenWeatherMap API 獲取天氣數據。

會員系統全面升級 Ver 4.0

usercenter 升級

更簡潔/擴展了會員中心(資訊&點播/福利)的互動功能。

登入系統 UI 重寫

重寫 LOGIN/signup/forget_pw 的UI - 升級Ver 2.0，更現代化。

儀表板整合

把本來在會員儀表板(點播/福利)的功能整合到統一的點播/福利儀表板。

抽運籤遊戲

新增一個抽運籤遊戲，增加用戶互動樂趣。

會員等級系統

增加會員等級系統(高級/VIP等等)，提供差異化服務。

伺服器狀態檢查

新增伺服器狀態檢查功能，實時監控系統運行狀態。
